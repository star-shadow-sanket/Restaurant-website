# 🍛 Kolkata Royal Spice Website

A simple **Restaurant Website** built using **HTML, CSS, and JavaScript**.
This project represents a restaurant web page where users can explore the **menu, reserve a table, view upcoming events, and find contact information**.

The website theme is inspired by **Experience Authentic Bengali Flavours**.

---

## 🚀 Features

* Modern restaurant landing page
* Food menu section with popular Bengali dishes
* Table reservation system
* Upcoming restaurant events
* Contact information section
* Responsive grid layout for menu
* Simple JavaScript validation for reservations

---

## 🛠 Technologies Used

* **HTML5** – Website structure
* **CSS3** – Layout and styling
* **JavaScript** – Reservation functionality

---

## 📂 Project Structure

```text
restaurant-website/
│
└── Restaurant.html
```

The project is built in **a single HTML file with embedded CSS and JavaScript**.

## 📅 Reservation System

Users can reserve a table by entering:

* Name
* Number of people
* Reservation date

JavaScript validates the input and displays a **confirmation message**.
